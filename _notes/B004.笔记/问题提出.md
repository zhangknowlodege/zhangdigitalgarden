# 超注意症状是否是精神分裂症患者执行功能损伤的内在原因？

- ## 第一段：介绍大背景：认知损伤关系模糊
	- 认知功能损伤是精神分裂症患者的一个核心症状，已被大量研究所证实（[[2021_张颖_精神分裂症认知功能损_KEY-VB8MA6DL]]；[[2020_McCutcheon_Schizophrenia-An Ove_KEY-RFE89EPU]]；[[2016_Owen_Schizophrenia_KEY-KFSA74JY]]；[[2022_Zhou_Working memory defic_KEY-ET2GCU24]]；[[2022_Wu_Imbalance Between Pr_KEY-PVUGZ8TM]]；[[2011_Lesh_Cognitive control de_KEY-QX6GBIPJ]]）
		- 不过，目前对于精神分裂症患者的认知缺陷究竟是特异性（specific ）的还是一般化（generalized）一直备受争论（[[2018_Gold_Selective Attention,_KEY-ZHZX5UIU#^KEY48QHMQP5]]；[[2009_Dickinson_Systemic hypotheses _KEY-7UU69JBG]]；[[2008_Carter_Identifying cognitiv_KEY-BGHU7GX8]]；[[2013_Green_Has the generalized _KEY-5YWIMXGD]]；[[2013_Gold_“Generalized cogniti_KEY-WH3IN96F]]）
			- 也就是说精神分裂症患者究竟特定认知功能损伤还是所有认知功能都损伤，或者说各认知子成分之间的损伤关系目前仍是未知。
				- 该问题的回答对于精神分裂症患者的认知损伤的机制解释以及进一步认知损伤干预极为关键（[[2019_Luck_Is attentional filte_KEY-XE7NVS5A]]；[[2018_Gold_Selective Attention,_KEY-ZHZX5UIU#^KEY4RM659XJ]]）。
- ## 第二段：铺垫问题一
	- 具体来看，大量研究发现精神分裂症患者的选择性注意、工作记忆、执行功能损伤（[[2011_Lesh_Cognitive control de_KEY-QX6GBIPJ]]）。不过由于这些认知功能与概念的重叠，使得我们难以理解这些认知子成分究竟是不同的认知功能损伤还是某种高级认知功能损伤的不同表现（[[2018_Gold_Selective Attention,_KEY-ZHZX5UIU#^KEYD27Z7J9J]]）。
	- 其中，执行功能是人脑多种复杂认知功能的集合，对个体的心理与行为起到了重要的控制于调节作用（[[2013_Diamond_Executive Functions_KEY-KFSKJ54A]] ）。
	- 目前来自行为（）、脑电（）、脑成像（）的证据都表明精神分离症患者的执行功能存在存缺陷。
		- 不过执行功能这一概念较为复杂，并非单一的成分，因而目前对于精神分裂症患者执行功能的损伤机制仍是未知。
		- 其中，目前对于精神分裂症的执行功能损伤究竟来自执行功能损伤本身还是来自于较低水平的认知损伤这一核心问题仍是未知（[[2022_Thuaire_Executive deficits i_KEY-XS5TDSLC#^KEYLHK4DZ3I]]）。
			-  例如，精神分裂症患者的执行功能缺陷是否是其选择注意缺陷导致的还是执行功能本身的功能缺陷？
			- 该问题的解决对于精神分裂症患者执行功能损伤的内在机制及其进一步干预极为关键。（意义）
- ## 第三段：引出问题一
	- 近些年，临床精神病患者（如自闭症、精神分裂症、注意多动缺陷障碍等）身上所发现的超注意（hyperfocusing）现象引起了研究们的诸多关注（[[2021_Ashinoff_Hyperfocus：The forgo_KEY-HJ47FVPM#^KEYVLW7MK6G]]），有学者认为该症状可能与临床精神病患者的认知缺陷密切相关（）。
		- 对于精神分裂症患者来说，传统观点一直认为精神分裂症患者存在注意控制缺陷（[[1979_Frith_Consciousness, infor_KEY-FJ9CNPAH]]），因而后期有大量研究者围绕注意控制缺陷与其认知损伤的关系。不过近些年提出了相反的观点，认为精神分裂症患者是存在超注意症状。
			- 例如，部分研究针对工作记忆容量,...但...... <mark style="background: #ADCCFFA6;">有学者尝试考察超注意现象来解释视觉工作记忆损伤</mark>....[[2014_Gray_Relationships Betwee_KEY-PXZGEBPB]]
			- 最近，luck归纳总结.....提出了 hyperfocusing hypothesis ，认为精神分裂症患者的认知损伤是由于其认知资源过度狭隘而专注造成的，也就是说HF是导致精神分裂症患者认知功能损伤的原因。
				- 他们还提出，解决HF在未来或许是改善认知损伤的一个突破口。
			- 不过，而精神分裂症患者的工作记忆容量与执行功能缺陷是完全独立的（[[2018_Gold_Selective Attention,_KEY-ZHZX5UIU]]）。
				- 那么，HF与精神分裂症患者的执行功能损伤存在什么关系？

- ## 第四段：引出研究问题二
	- 精神分裂症是一组病因未明的慢性精神疾病，其病程发展大致可分为发病前期、前驱期（也被称为临床高危阶段）、首发期（first-episode）和慢性期（chronic）四个主要阶段。
		- 针对精神分裂症患者病症研究长期存在一个争论，即精神分裂症究竟是神经退行性（Neurodegenerative）疾病还是神经发育性（Neurodevelopmental）疾病（[[2016_Owen_Schizophrenia_KEY-KFSA74JY#^KEYYS32663Y]]），
			- 与之相对应的就是，精神分裂症患者的认知损伤究竟是随其病程发展缓解还是加剧？
				- 由于该问题对于精神分裂症患者的早期筛查与干预至关重要，
					- 进而近些年的大量研究侧重于精神分裂症早期病症以及早期与后期病症的对比研究（[[2016_Owen_Schizophrenia_KEY-KFSA74JY#^KEYAUB8UDDG]]）。
			- 最近有研究发现，首发期精神分裂症患者的执行功能损伤会随期病程发展进一步恶化（[[2019_Zanelli_Cognitive Change in _KEY-56L5JJG5]]），
				- 该观点。
					- 那么，执行功能损伤加剧的内在原因是什呢？
						- 是否是精神分裂症患者超注意症状的加剧所造成的呢？
	- 虽然对于精神分裂症分裂症患者存在HF已被广泛证实，不过目前尚未有研究考察该症状随病程发展的变化过程。
		- 超注意是新发现的现象，其在病程发展过程中的变化仍是未知
			- [[2019_Luck_The hyperfocusing hy_KEY-P8FAQQR4#^KEYE3CR7PZ6]]
		- 因此，我们进一步考察HF与EF关系在精神分裂症患者病程中的变化情况，或许这能够帮助我们理解精神分裂症患者认知损伤随病程发展而严重，那么HF能够成为SZ早期筛选、干预的角度。

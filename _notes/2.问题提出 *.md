- # 基于脑电信号的运动想象
	- [ ] 前提
		- [ ] 高原环境下无论是静息态还是任务态（尤其是mental rotation， 它和MI极为相似）的脑电信号是变化的
		- [ ] 而BCI是依赖于MI的EEG信号的，如果低氧环境下这个信号是受损的，那么还能够识别运用吗
		- [ ] 平原和高原进行“mental rotation”的脑电是不同的， mental rotation和MI是极为相似的
	- [ ] 大脑进行 MI 活动时会产生 各种形式的信号，可通过不同手段测量
	- [ ] 其中，脑 电（electroencephalogram，EEG）测量的是大脑产生 的头皮电活动，具有无创、高时间分辨率、低成本 等特点
	- [ ] 而基于运动想象脑电（motor imagery EEG，MI-EEG）信号的脑-机接口（brain-computer interface，BCI）可以帮助用户在大脑与外部设备间 建立直接通道，完成操纵双臂机器人
	- [ ] 而长久以来，研究发现，高原低氧会影响人类的脑电信号
	- [ ] 那么在高原低氧环境进行运动想象和在平原进行运动想象的脑电信号是否存在差异？
	- [ ] 运动想象本质上也是一个认知活动
		- [ ] 个体在平原进行某项认知活动的脑电信号和在高原进行时是不同的
	- [ ] 是基于对脑电信号的识别，从而达到控制，但是脑电信号是受损的，那么还能够识别吗？
	- [ ] 测量
		- [ ] 在平原进行运动想象，然后记录其脑电信号
		- [ ] 在不同海拔高度（不同低氧状态Spo2来衡量），以考察
	- [ ] 低氧和运动想象、运动执行的关系
